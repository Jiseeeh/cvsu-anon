CvSU Anon.

A web application, similar to NGL, includes a feed where users can see which person (his/her username) has received the most messages. Users have a point system called 'ray points(RP)' where they earn points for each message they receive and send. To prevent users from sending messages to themselves by creating multiple accounts, each user is limited to having only one account per IP address.
